## Purpose
The purpose of this project is to have a first contact with the MERN stack by creating an online game known as "Wordle"

## Requeriments
- NodeJS v18 or later
- A MongoDB database
<a name="technologies"></a>
## Technologies
- **Database**: MongoDB
- **Database Server**: MongoDB Atlas
- **Back-End**: NodeJs and ExpressJS
- **API**: APIRest
- **Front-End**: React
